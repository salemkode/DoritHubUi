# DoritHubUi

## How to build from source :

```
git clone https://github.com/superastorh/DoritHubUi.git

cd DoritHubUi

npm install # or yarn install

npm run build

npm run start

```

## How to run for dev :

```
git clone https://github.com/superastorh/DoritHubUi.git

cd DoritHubUi

npm install # or yarn install

npm run dev
```
